# Pausa

**Pausa** es una metodología y plataforma de gestión consciente que ayuda a personas, equipos y organizaciones a trabajar mejor, en equilibrio y en paz. Combina tecnología reflexiva con consultoría para clarificar roles, tomar consciencia sobre el uso del tiempo y liderar a través de datos.

🌐 Sitio web: [www.pausaco.com](https://www.pausaco.com)

## Sobre este repositorio

Este repositorio contiene el sitio web estático de Pausa (marketing/landing), disponible en español e inglés.

## Estructura

```
.
├── index.html                     # Página principal (ES)
├── nosotros.html                  # Equipo y filosofía (ES)
├── reconocimiento.html            # Reconocimientos (ES)
├── terminos-y-condiciones.html    # Términos y condiciones (ES)
├── politicas-de-privacidad.html   # Política de privacidad (ES)
├── en/                            # Versión en inglés de todas las páginas
└── assets/                        # Estilos, scripts, imágenes y fuentes
```
