## Resumen de Uso

- **Color de marca dominante:** `#4A4BFF` — azul-violeta usado en logo, títulos y CTAs
- **Fondos:** `#FFFFFF` y `#F9F9F9` para secciones claras; `#4A4BFF` para secciones en azul
- **Texto:** `#0B1F34` para texto oscuro principal; `#4949E8` para texto sobre fondo blanco
- **Acentos cálidos:** terracota (`#F2EAE7`, `#E5CEC0`) y ámbar (`#FFB422`) para equilibrar el azul
- **Acento suave:** lavanda (`#D2D2FF`) como color complementario del violeta
